Allocations

Allocations let you distribute resources among participants based on their preferences. Click the Allocation tab in the sidebar. You can create instances, view ones you've created, and see ones you've been invited to participate in. Use the search box to filter by name.

Creating an Allocation

Allocation creation follows the same four steps as Poll creation. The key differences are in Step 1 and Step 4.

Step 1 — Basic Info
  • Set Question type to Allocation instead of Poll.
Step 2 — Alternatives & Step 3 — Invite Users

Same as regular poll creation. See the Polls guide for details.

Step 4 — Algorithms & UI
  • Algorithms — Only one allocation algorithm is currently implemented.
  • UI for preferences — Choose one preference UI. The three UIs cannot co-exist in the same instance (radio buttons, not checkboxes).
Allocation Settings

The Allocation settings pages are similar to Poll Settings — you can manage preferences, configure emails, and duplicate or delete the instance.
