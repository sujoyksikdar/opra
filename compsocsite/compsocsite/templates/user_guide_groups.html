Groups

Groups let you bundle users together so you can add them to polls and allocations all at once. Click the Groups tab in the sidebar to see and manage your groups. If a group is open, any OPRA user can join it. If it's closed, only the creator can add members.

Creating a Group
  • 1Click the "+" icon on the Groups page.
  • 2Give the group a name and set whether it is open or closed.
  • 3Search for and add members to the group.
Managing Members
  • After creation, open the group detail page to add or remove members at any time.
  • Only the creator of the group can add or remove members from a closed group.
Using Groups in Polls & Allocations

When creating or editing a poll or allocation, go to the Invite Users step and use the "Add/Remove groups of users" section to add or remove an entire group in one click.

ⓘ Adding a group adds all current members of that group to the poll at that moment. Future changes to the group's membership are not automatically reflected in existing polls.
